# Jeu ShiFuMi (Python - Console)

## Description
Ce projet est une version console du jeu ShiFuMi (Pierre-Papier-Ciseaux).  
Le joueur affronte l'ordinateur pendant plusieurs manches et le premier à atteindre 3 victoires gagne la partie.

## Objectif
J'ai réalisé ce projet pour m'entraîner à :
- utiliser des boucles et conditions complexes
- gérer des scores et variables
- créer une interaction entre joueur et ordinateur
- utiliser des nombres aléatoires pour simuler l’ordinateur

## Fonctionnalités
- Système de score pour le joueur et l’ordinateur
- Jeu sur plusieurs manches
- Indications claires pour les choix du joueur
- Gestion de choix invalides
- Affichage du vainqueur à la fin

## Fonctionnement
1. Le joueur choisit entre Pierre, Papier ou Ciseaux
2. L’ordinateur choisit aléatoirement son coup
3. Le résultat du tour est affiché (gagné/perdu)
4. Le jeu continue jusqu’à ce qu’un joueur atteigne 3 victoires
5. Le score final est affiché

## Technologies utilisées
- Python
- Module `random`
- Module `time` pour pause dramatique
